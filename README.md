# Bird Species Detection Project

This project is prepared for the **SENG 461 - Web Programming with Python and Git** course.

## Project Description
The aim of this project is to detect bird species using a **YOLOv8** AI model. The web application integrates the AI model with a **Flask** backend to process images and display results.

## Technologies Used
* **Frontend:** HTML, CSS
* **Backend:** Python, Flask
* **AI Model:** YOLOv8 (Computer Vision)
